# Arshal V | Portfolio Website

A modern, high-fidelity Bento Grid portfolio designed for Muhammed Arshal V, a Full Stack Developer from Kerala, India.

## Features
- **Sleek Bento Grid Layout**: Organized grid display highlighting technical skills, core details, and projects.
- **Independent Sticky Sidebar**: Desktop layout features a sticky dark sidebar that tracks along with right column scrolling.
- **Frameworks & Tooling Categorization**: Clearly displays frontend stacks, backend database setups, development frameworks, and workflows.
- **Client-side Avatar Upload**: Upload your own profile photo directly through the web interface with persistent browser `localStorage` caching.
- **Responsive Theme**: Tailored and fully fluid experience spanning desktops, tablets, and mobile screens.

## Technical Stack
- **Structure**: HTML5
- **Styling**: Tailwind CSS CDN & Custom CSS Variables
- **Icons**: Material Symbols Outlined
- **Logic**: Vanilla JavaScript
- **Host / Build**: Vite / Static HTML

## Setup & Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/arshal-salu/arshal-sallu.git
   ```
2. Open `index.html` directly in the browser, or serve it using a local development server:
   ```bash
   npx serve .
   ```
